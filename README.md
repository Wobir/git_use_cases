# создание SSH подключения
SSH - предпостительный протокол для работы с удаленными репозиториями поэтому нужно уметь создавать ключи и помещать на удаленный репозиторий  
git Bash(win 10 и win 11) или Терминал на Linux: __ssh-keygen__
если старая весрия windows(почему-то): необходима программа __Putty__.
при использовании ssh-keygen в папке пользователя появляется скрытая папка __.ssh/__ в ней будут находится файлы ключей. они создаются парами различаются расширением __.pub__. Файл с таким расширением мы можем передавать безопасно на нужные сервера/пк. Открываем файл блокнотом видим примерно такую запись: __"ssh-ed25519 AAAAC3NzaC1l1NTE5AAAAIt0AxO2uFif0F42RYBr+9HnXJpBgCJ9T teacher_wobir@ws-r-teacher"__ - Это публичная часть ключа её нужно скопировать и поместить в настройках аккаунта в блок **SSH and GPG keys**
