# создание локального репозитория и подключение к удаленному репозиторию
git init folder-name*  * указать если нужно поместить в директорию с нужным названием 
git remote add origin  https://сервер.удаленного.репозитроия:пользователь/названиерепозитория.git ПРИМЕР: git remote add origin https://github.com/Wobir/git_use_cases.git
git push -u origin master

(Теперь при заходе в эту папку можно будет делать коммиты и отправлять на удаленный репозиторий.)
